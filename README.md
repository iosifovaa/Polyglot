# Polyglot

Polyglot — это современная платформа для изучения иностранных языков через сериалы, музыку, короткие видео, грамматику и интерактивные задания.

## Основная идея проекта

Сайт помогает изучать язык через современный контент и реальные диалоги, а также предлагает персонализированное обучение в зависимости от целей пользователя.

## Основные функции

- Выбор языка интерфейса
- Регистрация пользователя
- Персональная анкета
- Уроки по темам
- Грамматика
- Изучение через видео и музыку
- Интерактивные тесты
- Личный словарь
- Прогресс пользователя
